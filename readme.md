# 專案目的

這是一個使用 .NET 10 Blazor 開發框架所建立的專案，旨在提供一個快速、預製、方便的使用 Blazor 來構建現代化的 Web 應用程式。其特色為

* 採用伺服器端渲染，且是全域性而不是採用元件為主

以下是整理完成後的版本，可直接放到 `README` 或專案說明文件中：

## 專案目的

本專案基於 `.NET 10` 與 `Blazor` 建立，目標是提供一個**可快速啟動、預先配置、易於擴充**的現代化 Web 應用程式樣板，協助開發者以更低成本開始實作商業功能。

## 特色

- [x] 採用伺服器端渲染（SSR），提升初始載入速度與 SEO 友善性。  
- [x] 使用**全域渲染策略**進行設定，而非以單一元件逐一指定，讓架構一致、維護更單純。  
- [x] 以 Blazor 為核心，保留 `.NET` 生態系的型別安全、開發效率與可測試性。
- [x] 採用 NLog 6.1 作為日誌輸出裝置
- [x] UI 元件則優先採用 AntDesign 元件庫，提供豐富且現代化的使用者介面元素。
- [x] 預先建立必要的專案：商業邏輯、資料模型、資料存取、共用等等
- [x] 預設採用 EF Core for SQLite 資料庫
- [x] CRUD 基本設計與搜尋、篩選和分頁機制
- 使用者角色紀錄 CRUD 處理
- 使用者紀錄 CRUD 處理
- 使用者角色管理機制
- [x] Web API 設定
- Web API CRUD
- [x] Swagger UI 
- [x] 使用者登入與登出之驗證與授權機制
- [x] 靜態檔案對應機制
- [x] 加入 Cookie 身分驗證的宣告
- 加入 JWT 身分驗證的宣告
- Sidebar 導覽列與清單定義JSON檔案
